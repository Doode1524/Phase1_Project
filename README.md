# Phase1Project

"WELCOME TO LEGEND!!"
 "Legend is an app that gives base statistics on Champions in the MOBA game League of Legends!"

 # Installation

gem 'rspec'
gem 'pry'
gem 'require_all'
gem "rake", "~> 12.0"
gem "json"
gem "open-uri"
gem "bundler"
gem 'rest-client'

And then execute $ bundle install

run $ 'ruby bin/run' to start
